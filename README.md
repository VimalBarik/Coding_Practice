# Coding Practice

A collection of practice files I've written since 2021, covering everything 
from programming fundamentals to machine learning. These are not projects — 
they are notes, experiments, and exercises written while learning.

## Contents

### Fundamentals
- **C** — basics from early programming days
- **SQL** — queries, joins, aggregations
- **HTML** — basic structure and markup
- **Python** — core Python, syntax, and language features

### Python Deep Dives
- **Python DSA** — data structures and algorithms (arrays, trees, graphs, etc.)
- **Python OOP** — classes, inheritance, decorators, generators

### Data & ML
- **Pandas** — dataframes, cleaning, EDA workflows
- **Statistics** — descriptive stats, probability, distributions
- **Data Preprocessing** — encoding, scaling, pipelines
- **Scikit-learn** — classification, regression, model evaluation
- **TensorFlow** — tensors, CNNs, RNNs, transfer learning, tf.data

### Projects
A few standalone `.py` files from small projects built during practice.

## Note
Files were created between 2021 and 2025. Most ML and data science work 
is in `.ipynb` notebooks — outputs are preserved so they can be read 
directly on GitHub without running them.
